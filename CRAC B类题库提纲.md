# 法律法规类

## 一、核心管理法规与制定机构

1. **全国无线电管理行政法规**（0001 题）
    - 法规名称：《中华人民共和国无线电管理条例》
    - 制定机构：国务院和中央军委
    - 排除：《中华人民共和国无线电管理办法》（错误）、工业和信息化部（错误）
2. **业余无线电台专项管理文件**（0002 题）
    - 管理文件：《业余无线电台管理办法》
    - 制定机构：工业和信息化部
    - 排除：《业余无线电台管理暂行规定》（错误）、国务院（错误）
3. **业余业务术语定义法规**（0004 题）
    - 文件名称：《中华人民共和国无线电频率划分规定》
    - 定义内容：“业余业务”“卫星业余业务”“业余电台”
    - 排除：《无线电管理条例》《电信条例》（错误）
4. **新版《业余无线电台管理办法》施行时间**（0005 题）
    - 施行日期：2024 年 3 月 1 日
    - 排除：2024 年 12 月、2025 年 3 月 / 12 月（错误）

## 二、业余电台管理核心规则

1. **业余无线电爱好者定义**（0015 题）
    - 核心条件：经正式批准、对无线电技术有兴趣、兴趣纯个人爱好（不涉及谋取利润）
    - 排除：可涉及利润（错误）、仅限成年人（错误）
2. **业余电台设置与设备要求**（0027-0029 题）
    - 个人设台材料：申请表、身份证明（未成年人需监护人材料）、设备说明材料
    - 单位设台材料：申请表、设备说明材料、单位资质 + 技术负责人说明
    - 设备要求：商品设备需型号核准（含业余频段）；自制设备需符合国标 + 仅限业余频段
3. **业余电台执照管理**（0033-0035 题）
    - 有效期：不超过 5 年
    - 执照形式：纸质 / 电子同等效力，国家统一样式
    - 旧执照效力：新版《办法》施行前执照，有效期内按原参数使用；2012 年版《办法》同时废止
    - 排除：终身有效（错误）、2025 年 3 月施行新版（错误）
4. **终止使用业余电台手续**（0043-0044 题）
    - 注销机构：作出许可决定的无线电管理机构
    - 后续要求：交回执照，注销之日起 60 个工作日内拆除设备
    - 排除：6 个月内拆除（错误）、仅交回执照（错误）

## 三、操作能力验证与资质

1. **操作能力验证机构**（0050-0051 题）
    - A/B 类验证：国家 + 省、自治区、直辖市无线电管理机构
    - C 类验证：仅国家无线电管理机构
    - 排除：地方协会、教育机构（错误）
2. **验证收费与证书**（0052-0053 题）
    - 收费规则：不收取考试费用
    - 有效证书：无线电管理机构颁发的 “业余无线电台操作技术能力验证证书”
    - 排除：协会颁发证书（错误）、按电台类型收费（错误）
3. **操作能力分类与权限**（0055-0059 题）
    - 分类：A 类、B 类、C 类（无 D 类）
    - 核心区别：允许工作的频率范围、最大发射功率
    - 举例：
        - A 类：30-3000MHz，≤25 瓦
        - B 类：30MHz 以下＜15 瓦 / 30MHz 以上≤25 瓦
        - C 类：30MHz 以下≤1000 瓦 / 30MHz 以上≤25 瓦

## 四、呼号使用规范

1. **呼号核发与重分配**（0072-0074 题）
    - 核发时机：颁发业余电台执照时同时核发
    - 重分配规则：非中继 / 信标呼号注销 1 年后可重分配；原申请人再次申请优先核发原呼号
    - 排除：受理机构核发、需单独申请（错误）
2. **呼号构成（国内）**（0077-0081 题）
    - 四部分：前缀（国际分配字母 B，1 位）、电台种类（G/H/I 等一般台；J 空间台；R 中继 / 信标台）、分区编号（1 位数字）、后缀（1-4 位字母 / 字母 + 数字，特定组合保留）
    - 排除：前缀多字母（错误）、后缀纯数字（错误）
3. **呼号使用要求**（0079-0080 题）
    - 必发场景：通信建立 / 结束时发送，通信中间隔≤10 分钟发送
    - 禁止：用姓名、代号代替完整呼号

## 五、法律责任与违规处罚

1. **刑事处罚依据**（0134 题）
    - 法规：《中华人民共和国刑法》
    - 情形：擅自设台、用频率干扰通讯秩序，情节严重 3 年以下，特别严重 3-7 年（单位犯罪追责主管人员）
2. **频谱资源权属**（0135 题）
    - 法规：《中华人民共和国民法典》（第 252 条）
    - 原则：无线电频谱资源属于国家所有
3. **无线电管制**（0136-0140 题）
    - 定义：特定时间 + 特定区域内，对无线电波发射、辐射、传播的强制性管理
    - 决定机构：全国 / 跨省由国务院 + 中央军委；省范围内由省人民政府 + 相关军区
    - 违规处罚：无线电管理机构（责令改正、拆设备、吊销执照）、公安机关（治安处罚）
4. **常见违规处罚**（0142-0149 题）
    - 擅自用频率 / 设台：责令改正，没收设备 + 违法所得，5 万以下罚款；拒不改 5-20 万
    - 不按执照用台 / 故意收非许可信号：责令改正，3 万以下罚款；严重吊销执照 3-10 万
    - 干扰人身安全频率（船舶、航空器等）：20-50 万罚款
    - 干扰无线电业务：拒不改没收设备，5-20 万，吊销执照
5. **治安管理处罚**（0158 题）
    - 情形：故意干扰无线电业务 / 产生有害干扰，经指出拒不消除
    - 处罚：5-10 日拘留；情节严重 10-15 日拘留

## 六、业余电台禁止行为

1. **核心禁止事项**（0123-0124 题）
    - 禁止内容：
        - 发送与业余业务无关的信号（如广播、电话通话）
        - 利用电台牟利、传播违法 / 违反公共道德信息
        - 擅自编呼号、涂改 / 倒卖执照
        - 传播、公布无意接收的信息
        - 干扰其他合法电台通信🔶1-602 至 1-608、1-669 至 1-681🔷
2. **应急通信特殊规则**（0127-0130 题）
    - 允许与非业余台通信：仅突发重大自然灾害等紧急情况，内容限于抢险救灾 / 应急任务
    - 临时设台：危及安全时可临时设台，48 小时内报告无线电管理机构，紧急消除后关闭

## 三、电波传播核心特性

### 3.1 各频段传播方式对比

|频段类型|传播主导方式|影响因素|通信距离范围|对应题目|
|---|---|---|---|---|
|LF/MF（135kHz、1.8MHz 等）|地波为主，天波为辅|大地电导率、地形障碍|近距离（数公里至数十公里）|0215、0217 题|
|HF（3.5-29.7MHz）|天波（电离层反射）为主|太阳活动（黑子、耀斑）、电离层层级（F1/F2 层）|中远距离（数百至数千公里）|0584、0688 题|
|VHF/UHF（50-440MHz）|视距传播（直射波）为主|天线高度、地形遮挡、多径效应|近距离（数公里至百公里）|0664、0668 题|
|卫星业余业务|空间直射波|卫星轨道、极化匹配、多普勒频移|全球范围（经卫星转发）|0274、0294 题|

### 3.2 电离层对 HF 传播的影响

- **关键层级**：
    - F2 层：主要反射 HF 频段电波，白天 / 夜间均起作用，是远距离 HF 通信的核心层级
    - E 层：辅助反射，突发 E 层（Es 层）可实现 VHF 超视距传播（如 6 米、2 米波段）（0670、0672 题）
- **影响因素**：
    - 太阳活动：太阳黑子数增多、耀斑爆发时，电离层电离度增强，HF 传播距离更远
    - 时段：白天 F 层电离度高，适合较高 HF 频段（如 20 米、15 米）；夜间适合较低 HF 频段（如 80 米、160 米）（0286 题）

## 四、特殊传播现象与应对

### 4.1 常见传播现象及影响

|传播现象|产生原因|影响频段|对通信的影响|对应题目|
|---|---|---|---|---|
|多径效应|电波经直射、反射（地面 / 建筑物）等多路径到达接收端|VHF/UHF 为主|信号强度波动、图像重影（ATV）、误码率上升|0664、0667 题|
|大气波导|高空逆温导致大气折射率突变，电波被 “束缚” 在波导层传播|VHF/UHF|实现超视距传播（数百公里）|0669、0671 题|
|突发 E 层（Es 层）|电离层 E 层局部强烈电离，形成反射层|6 米、2 米波段|短时间内实现上千公里 VHF 超视距通信|0670、0672 题|
|多普勒频移|收发双方相对运动（如卫星、移动台）|卫星通信、VHF 移动台|接收频率偏移，需调整 RIT/XIT 补偿|0294、0323 题|

### 4.2 传播异常应对措施

- **多径效应**：移动接收时调整位置、选用定向天线；固定接收时采用分集接收（0602 题）
- **频率偏移**：通过收发信机 “RIT（接收增量调谐）”“XIT（发射增量调谐）” 微调频率（0586、0587 题）
- **信号衰落**：HF 通信时选择合适频段（如夜间换低 HF 频段）、VHF/UHF 通信时提高天线高度（0688、0668 题）

## 五、频段使用细节规范

### 5.1 中继台频差要求

|中继台频段|上下行频差|备注（避免占用频率）|对应题目|
|---|---|---|---|
|144MHz 频段|600kHz|避免占用 144-144.035MHz（信标 / 数据）、145.8-146MHz（卫星）|0282、01059 题|
|430MHz 频段|5MHz|避免占用 431.9-432.240MHz（数据）、435-438MHz（卫星）|0282、01064 题|

### 5.2 避免干扰的频率范围

- **信标台频率**：短波业余电台应避开 IARU 信标频率 ±500Hz，信标频率为 14.100MHz、18.110MHz、21.150MHz、24.930MHz、28.200MHz（0185 题）
- **卫星专用频段**：
    - 10 米波段：29.3-29.51MHz（卫星业余业务，话音不占用）（0228 题）
    - 2 米波段：145.8-146MHz（卫星业余业务，话音不占用）（0229 题）
    - 70 厘米波段：435-438MHz（卫星业余业务，话音不占用）（0230 题）
- **CW/SSB/FM 频率划分**：
    - 7MHz 频段 LSB 通话：7.030-7.200MHz（0198 题）
    - 14MHz 频段 USB 通话：14.100-14.350MHz（0199 题）
    - 29MHz 频段 FM 通话：29.51-29.7MHz（0204 题）

### 5.3 30MHz 以下频段（短波）B 类操作特殊规范

1. **功率限制实操**（ 0283 题）
    
    - B 类在 30MHz 以下频段发射功率需严格控制在**15 瓦峰值功率**以内，实操中需通过功率计实时监测 SSB 调制信号（峰值功率约为平均功率 4 倍，故平均功率需≤3.75 瓦），避免瞬时超标
    - 禁止通过关闭 ALC 电路、修改设备参数等方式突破功率限制
2. **频段细分使用规范**（ 0284 题）
    
    - 1.8MHz（160 米）：日出 / 日落时段避免大功率发射，优先保障远距离弱信号通信，不得占用 1.800-1.810MHz CW 专用子段
    - 3.5MHz（80 米）：国内区域通信为主，夜间天波传播时需避让广播业务，话音集中在 3.700-3.900MHz
    - 7MHz（40 米）：国际通用段，7.000-7.030MHz 为 CW / 数据专用，SSB 话音需在 7.030MHz 以上，避免占用卫星信标频率

## 三、各类别验证报考条件

### 3.1 基础报考要求（通用）

- 核心前提：熟悉无线电管理规定、具备对应操作技术能力（0062、0322 题）
- 年龄限制：无强制成年要求（未成年人可报考，操作不受年龄限制，需监护人材料）（0036、0037 题）

### 3.2 分类别专项条件

### 5.1 旧证书过渡政策（新版《办法》施行前）

### 5.2 未成年人操作权限

- 设置权限：未成年人可申请设置规定条件内的电台（如 A 类可设≤25 瓦 VHF 电台，B 类同）（0037、0361 题）
- 操作权限：具备《操作证书》者独立操作不受年龄限制（0036 题）

### 5.3 实习操作要求（未取得证书人员）

- 监督要求：需由电台设置人 / 技术负责人现场监督指导（0583 题）

# 呼号使用规范

## 一、呼号核发与分配规则

### 1.1 核发时机与主体

- **核发节点**：无线电管理机构颁发业余无线电台执照时，**同时向申请人核发业余无线电台呼号**，无需单独申请（0072 题）
- **核发主体**：仅无线电管理机构，排除受理服务机构、业余无线电协会等（0072、0385 题）
- **重复核发限制**：若申请人已取得业余中继台、业余信标台呼号以外的其他业余无线电台呼号，无线电管理机构**不再核发新呼号**（0073 题）

### 1.2 注销与重分配规则

|场景|操作要求|时间限制|对应题目|
|---|---|---|---|
|呼号停止使用|需依法向作出许可决定的无线电管理机构办理注销|无强制时限（及时注销）|0074、0395 题|

|中继 / 信标呼号重分配|无明确重分配时间（特殊管理）|不适用 “1 年” 规则|0074 题|

### 2.1 呼号结构（四部分）

- **第二部分顺序**：各省、自治区、直辖市无线电管理机构按 “G、H、I、D、A、B、C、E、F、K、L” 顺序核发（0083 题）

# 通信操作类

## 一、呼叫规范（核心：完整呼号 + 正确流程）

### 1.1 通用呼叫原则

- **前置要求**：发起呼叫前需先守听，确认频率空闲；若不确定，可询问 “有人使用频率吗？”（英文：Is this frequency in use?），无应答再呼叫（0255、0306 题）
- **呼号要求**：必须使用完整呼号，禁止用姓名、代号、别名或呼号部分（如仅后缀）替代（0079、0459 题）
- **禁止行为**：不得在未确认频率空闲时直接呼叫，避免干扰他人（0255 题）

### 1.2 不同场景呼叫方式

|呼叫类型|操作流程|示例（话音 / CW）|对应题目|
|---|---|---|---|
|CQ 呼叫（非特指呼叫）|1. 话音：呼叫 “CQ CQ CQ，这里是 [本台呼号]，[呼号字母拼读]，呼叫 CQ，听到请回答”（英文：CQ CQ CQ, this is [呼号], standing by）<br><br>2. CW：发送 “CQ CQ CQ DE [本台呼号] [呼号重复 2 次] PSE K”|话音：CQ CQ CQ，这里是 BH1ZZZ，Bravo Hotel One Zulu Zulu Zulu，BH1ZZZ 呼叫 CQ，听到请回答<br><br>CW：CQ CQ CQ DE BH1ZZZ BH1ZZZ BH1ZZZ PSE K|0248、0295 题|
|特定台呼叫|1. 话音：重复对方呼号 2-3 次 + 本台呼号 + 字母拼读<br><br>2. CW：重复对方呼号 2-3 次 +“DE”+ 本台呼号（重复 2 次）+“PSE K”|话音：BH8YYY，BH8YYY，这里是 BH1ZZZ，Bravo Hotel One Zulu Zulu Zulu，BH1ZZZ 呼叫<br><br>CW：BH8YYY BH8YYY DE BH1ZZZ BH1ZZZ PSE K|0297、0298 题|
|回应 CQ 呼叫|1. 先报对方呼号，再报本台呼号（避免颠倒）<br><br>2. 清晰拼读关键字符，必要时重复|话音：BH1ZZZ，这里是 BH8YYY，Bravo Hotel Eight Yankee Yankee Yankee<br><br>CW：BH1ZZZ DE BH8YYY PSE K|0249、0317 题|

## 二、信号报告（统一 RST 格式，按需调整）

### 2.1 RST 格式含义（核心：R - 可辨度、S - 强度、T - 纯净度）

|字母|含义|评分范围（1-9）|适用场景|对应题目|
|---|---|---|---|---|
|R|信号可辨度|1（完全不可辨）-9（完全清晰）|所有调制方式（CW、SSB、FM）|0253、0319 题|
|S|信号强度|1（极弱）-9（极强）|所有调制方式|0253、0319 题|
|T|信号纯净度（失真度）|1（严重失真）-9（无失真）|仅 CW、SSB（FM 不考核 T 值）|0254、0319 题|

### 2.2 不同调制方式报告示例

|调制方式|报告格式|示例（对方呼号：BH1ZZZ）|对应题目|
|---|---|---|---|
|CW（等幅电报）|RST 完整报告（3 位数字）|话音：BH1ZZZ，你的信号 579<br><br>CW：UR RST 579|0254、0319 题|
|FM（调频话音）|仅 RS 报告（2 位数字，无 T 值）|话音：BH1ZZZ，你的信号 57<br><br>英文：BH1ZZZ, you are 57|0254、0311 题|
|SSB（单边带话音）|RST 完整报告（3 位数字）|话音：BH1ZZZ，你的信号 569<br><br>CW：UR RST 569|0254、0312 题|

### 2.3 报告交换目的

- 核心作用：双方通过报告了解电波传播情况，调整操作参数（如功率、频率）（0252 题）
- 禁止行为：不得随意给出虚假报告（如一律报 599），需真实反映信号质量（0252 题）

## 四、日志与 QSL 卡管理（合规 + 记录完整）

### 4.1 通联日志必要内容

- **核心要素**（缺一不可）：
    1. 通信时间（UTC 时间 / 本地时间 + 日期）
    2. 通信频率、通信模式（CW/SSB/FM 等）
    3. 对方呼号、双方信号报告
- **排除无关内容**：对方姓名、天气、设备型号（非强制记录）（0262、0263 题）
- **法律要求**：日志需妥善保存，作为通联证明（如申请奖状）（0262 题）

### 4.2 QSL 卡交换规则

|项目|要求|示例|对应题目|
|---|---|---|---|
|QSL 卡要素|1. 双方呼号；2. 通信时间、频率、模式；3. 双方信号报告；4. 操作员签章、通信地址|呼号：BH1ZZZ/BH8YYY；频率：14.200MHz；报告：579/579|0265、0375 题|
|交换场景|1. 确认通联（本地 / 国内 / 国际）；2. 申请奖状（如 WAZ、WAS）；3. 设备性能验证|申请 WAZ 奖状需提交不同 CQ 分区的 QSL 卡|0264、0266 题|
|特殊情况|收到短波收听者（SWL）的 QSL 卡，需回卡致敬|SWL 收听 BH1ZZZ 后寄卡，BH1ZZZ 需回寄|0269、0281 题|
|禁止行为|1. 空白 QSL 卡不得作为名片赠送；2. 未通联不得寄送 QSL 卡|不得向未通联的电台寄卡 “刷成绩”|0266、0278 题|

## 五、插入通信与应急处理（礼貌 + 合规）

### 5.1 插入通信（加入他人通联）

- **正确流程**：
    1. 在一方停止讲话时发送 “插入”（英文：Break），避免打断正在讲话的电台；
    2. 得到对方回应后，说明 “[本台呼号] 请求插入”；
    3. 待对方正式邀请后再对话（0256 题）
- **禁止行为**：
    1. 直接打断他人讲话插入；
    2. 无回应时反复加大功率尝试（0256 题）

### 5.2 遇险信号与紧急处理

- **遇险插入**：若听到遇险电台请求插入，需立即停止当前通联，优先与遇险台确认险情、提供帮助（0257 题）
- **干扰应对**：
    1. 遇违法 / 不明干扰：不予理睬，收集干扰时间、频率、信号特征等信息，向无线电管理机构举报；
    2. 禁止：在频率上与干扰者争执、用大功率压制干扰（0260 题）
- **操作不规范应对**：对方操作不规范但无严重干扰时，通过电话、邮件等私下提出建议，禁止在频率上当面指责（0259 题）

### 5.3 B 类电台应急响应规范

1. **预登记要求**（新增 0159 题）
    
    - 参与本地应急通信网的 B 类电台需在属地无线电管理机构备案，登记设备型号、固定应急频率（如 144.000MHz 本地应急信道）
    - 每年需参加 1 次应急演练，未达标者取消应急入网资格
2. **通信优先级规则**（新增 0160 题）
    
    - 应急期间遵循 "生命优先 > 抢险指挥 > 物资调度 > 普通通信"
    - 普通通联收到 "Emergency" 呼叫时，需立即停止发射并回应 "Stand by for emergency traffic"
    - 应急信号格式："紧急（Emergency）+ 本台呼号 + 险情位置 + 需求"，如 "Emergency, BH1XXX, flood at XX village, need rescue team"

## 六、特殊场景通信操作

### 6.1 中继台通联

- **频差设置**：144MHz 频段中继台上下行频差 600KHz，430MHz 频段频差 5MHz（0282 题）
- **呼号发送**：呼叫时需报对方呼号 + 本台呼号，避免仅报 “CQ”（0283 题）
- **使用原则**：中继台上下行频率需保持空闲，以备紧急呼叫，禁止长时间占用（0272 题）

### 6.2 DX 通联（远距离通联）

- **呼叫方式**：呼叫 DX 电台时可使用 “CQ DX” 标识，示例：“CQ DX CQ DX，这里是 BH1ZZZ，呼叫 DX 电台”（0523 题）
- **效率原则**：仅交换呼号、信号报告、QSL 信息，避免无关闲聊，不占用频率（0524 题）
- **网络通联**：参加 DX 网络时，需遵守网络规则，经主控台允许后登录，按安排应答（0525 题）

### 6.3 未成年人 / 实习操作

- **未成年人操作**：具备《操作证书》者独立操作不受年龄限制；未成年人设台需提交监护人材料（0036、0037 题）
- **实习操作**：未取得证书者需在电台设置人 / 技术负责人现场监督下操作，功率不得超过监督人证书权限（0583 题）
## 七、短波通信特有操作（B 类核心场景）

1. **频段选择策略**（新增 0584 题）
    
    - 太阳黑子高活动期：优先使用 21MHz（15 米）、28MHz（10 米）等较高频段，依赖 F2 层反射实现远距离通信
    - 太阳黑子低活动期：转向 3.5MHz（80 米）、7MHz（40 米）等较低频段，利用 D 层衰减小的特性
    - 时段调整：白天选 14MHz（20 米）及以上，夜间切换至 3.5MHz（80 米）、1.8MHz（160 米）
2. **QRP 操作技巧**（新增 0585 题）
    
    - 天线优化：使用 dipole 天线或端馈天线，缩短馈线长度（建议≤10 米）以减少损耗
    - 调制方式：采用 15-20WPM 慢速 CW 或窄带 SSB，提升信号辨识度；避免使用宽频带 FM 模式
3. **国际通联规范**（新增 0586 题）
    
    - 语言要求：优先使用英语，呼号拼读必须使用国际 phonetic alphabet（如 B-Bravo、H-Hotel）
    - 信息交换：仅传递呼号、信号报告、QTH（位置）及必要 QSL 卡交换意愿，避免无关内容
    - 跨境频率避让：中俄边境地区 3.5MHz 频段上限为 3.8MHz，中越边境 7MHz 频段需避开 7.000-7.050MHz（对方专用段）

# 天线与馈线类

## 一、天线基础（功能与类型）

### 1.1 天线核心功能

- 发射天线的作用是什么？
    
    答案要点：把发射机输出的射频信号转化为无线电波【0604】
- 接收天线的作用是什么？
    
    答案要点：把空间的无线电波转化为接收机中的射频电信号【0605】

### 1.2 常见天线类型特性

- 垂直天线的关键特性是什么？
    
    答案要点：1. 发射垂直极化波（电场方向与地面垂直）；2. 水平方向无指向性（H 面方向图为全向）；3. 典型类型为 1/4 波长垂直接地天线（GP 天线），需依赖地网形成 “镜像” 振子【0689】
- 水平极化偶极天线的关键特性是什么？
    
    答案要点：1. 发射水平极化波（电场方向与地面平行）；2. 水平面（E 面）方向图呈 “8” 字展开，垂直于振子方向辐射最强；3. 谐振时总电气长度为 1/2 波长的整数倍，输入阻抗较低【0690】
- 八木天线的结构与适用场景是什么？
    
    答案要点：1. 核心结构：1 根有源振子（半波长偶极天线）+ 多根引向器（增强前向增益）+ 1 根反射器（抑制后向信号）；2. 定向性强、增益高（3-15dBi）；3. 适用于 VHF/UHF 频段远距离点对点通信（如 50MHz、144MHz）【0744】
- 加感天线的设计与效率原则是什么？
    
    答案要点：1. 适用场景：空间受限无法架设全尺寸天线时，通过串联电感补偿振子缩短的感抗；2. 效率排序（相同振子长度）：顶部加感＞中部加感＞底部加感；3. 缺点：辐射效率低于全尺寸天线【0665】
- 车载 “橡胶天线” 的使用问题是什么？
    
    答案要点：1. 车体金属结构产生屏蔽，削弱信号强度；2. 垂直极化分量更强，但增益显著低于全尺寸垂直天线；3. 护套开裂易导致内部振子氧化，影响性能【0661】

## 二、天线增益与极化匹配

### 2.1 天线增益定义与单位

- 天线增益的核心定义是什么？
    
    答案要点：待测天线最大辐射方向的辐射功率密度，与基准天线（如理想点源、半波长偶极天线）对应值的比值，反映天线集中辐射能量的能力【0651】
- 以 “dBi” 为单位的天线增益，参考基准是什么？
    
    答案要点：理想点源天线（理论上能向所有方向均匀辐射的假想天线），半波长偶极天线的增益约为 2.15dBi【0652】
- 以 “dBd” 为单位的天线增益，参考基准是什么？
    
    答案要点：半波长偶极天线，规定其增益为 0dBd；dBd 与 dBi 换算关系：1dBd≈3.15dBi【0654】

### 2.2 极化匹配与应用

- 极化不匹配对通信的影响是什么？
    
    答案要点：1. 水平极化与垂直极化混用：信号衰减≥30dB，通信效果显著变差；2. 圆极化天线可兼容任意线极化信号，但线极化天线无法兼容圆极化信号【0691】
- 卫星通信中如何应对极化变化？
    
    答案要点：1. 卫星旋转会导致下行信号极化方向变化；2. 接收端应选用左旋或右旋圆极化天线，避免因极化不匹配导致接收中断【0706】
- 视距通联中，收发天线极化的最佳配置是什么？
    
    答案要点：1. 收发天线极化方式需一致（如均为水平极化、均为垂直极化）；2. 若发射为左旋圆极化，接收也需为指向发射点的左旋圆极化【0703】

## 三、馈线特性与选型

### 3.1 馈线核心参数

- 业余电台最适合的同轴电缆特性阻抗是多少？
    
    答案要点：50Ω，匹配业余电台收发信机输出阻抗及多数天线的输入阻抗【0678】
- 为何业余电台优先选用同轴电缆作为馈线？
    
    答案要点：1. 屏蔽性好，与周围物体互感低，减少干扰；2. 易于安装，适应多数架设环境；3. 特性阻抗稳定，匹配效果好【0679】
- 选购同轴电缆时需关注哪些关键参数？
    
    答案要点：1. 特性阻抗（需与电台、天线匹配，通常为 50Ω）；2. 指定频率下的传输损耗（频率越高损耗越大，长距离优先选粗线径电缆如 RG-8）；3. 屏蔽层结构（双屏蔽、高编织密度可提升抗干扰能力）【0680】
- 信号频率对同轴电缆损耗的影响是什么？
    
    答案要点：频率越高，传输损耗越大（高频信号在屏蔽层趋肤效应更明显，介质损耗增加）；例如 430MHz 频段损耗显著高于 144MHz【0685】

### 3.2 馈线故障与防护

- 受潮对同轴电缆的危害是什么？
    
    答案要点：1. 介质损耗增大，信号传输效率下降；2. 芯线与屏蔽层氧化、锈蚀，严重时导致断路；3. 特性阻抗不稳定，驻波比升高【0682】
- 室外架设同轴电缆为何需耐受紫外线？
    
    答案要点：紫外线会老化电缆护套，导致护套开裂、进水，进而引发内部受潮故障【0683】
- 空气介质同轴电缆的主要劣势是什么？
    
    答案要点：需特殊密封措施防止湿气渗透（空气介质吸水后介电常数变化，影响特性阻抗与损耗），且柔韧性差，不易弯曲架设【0684】

## 四、驻波比与匹配部件

### 4.1 驻波比（SWR）核心知识

- 驻波比的定义与理想值是什么？
    
    答案要点：1. 反映传输线终端负载阻抗与传输线特性阻抗的匹配程度，计算公式为 “最大电压 / 最小电压” 或 “最大电流 / 最小电流”；2. 理想值为 1:1（完全匹配），可接受范围为≤2:1【0670】
- 驻波比显示 4:1 意味着什么？
    
    答案要点：1. 天线系统与馈线匹配极差，大量射频能量被反射回发射机；2. 可能导致发射机功率管过载，需检查天线接头、馈线完整性或天线谐振状态【0671】
- 发射机为何在驻波比超限时降低输出功率？
    
    答案要点：保护发射机内部功率半导体器件（如功率管），避免反射功率导致器件过热损坏【0673】
- 驻波比不稳定的常见原因是什么？
    
    答案要点：1. 馈线接头松动、氧化（如 N 型、M 型接头接触不良）；2. 天线振子接触不良或形变；3. SSB 模式下信号幅度波动，影响驻波比测量稳定性（建议用 CW/FM 信号校准）【0675】

### 4.2 关键匹配部件

- 巴伦（BALUN）的作用与原理是什么？
    
    答案要点：1. 名称由来：“平衡（Balanced）” 与 “不平衡（Unbalanced）” 的缩写；2. 核心功能：在平衡电路（如偶极天线）与不平衡电路（如 50Ω 同轴电缆）间传递射频能量，阻断寄生耦合；3. 常见应用：偶极天线与同轴电缆的连接，抑制馈线辐射干扰【0728、0729】
- 天线调谐器（ATU）的功能是什么？
    
    答案要点：1. 通过可调电容、电感网络，补偿天线与馈线的阻抗差异；2. 适用场景：多频段天线切换、临时架设天线（如野外通联）时，将驻波比降至可接受范围；3. 注意：无法降低馈线本身的传输损耗【0621】
- 中继台双工器的作用是什么？
    
    答案要点：1. 实现中继台 “单天线共用”，隔离上行频率（接收）与下行频率（发射）；2. 核心结构：多组带通滤波器，防止发射信号干扰接收机；3. 关键指标：隔离度≥60dB，避免收发信号串扰【0553】

# 安全操作规范类

## 一、法律合规安全类

|无线电频谱资源属性（0132）|1. 国家所有；2. 统一规划、有偿使用；3. 可实施无线电管制|
|擅自设台 / 用频率处罚（0142/0143）|1. 责令改正，没收设备 / 违法所得；2. 5 万以下罚款；3. 拒不改正 5-20 万|
|无线电管制实施（0138/0139/0140）|1. 特定时间 + 区域；2. 全国 / 跨省：国务院 + 中央军委；3. 处罚：无线电管理机构 / 公安机关|

## 二、设备操作安全类

|题目核心|答案速记提纲|
|---|---|
|设备型号核准要求（0029/0111）|1. 商品设备需型号核准；2. 自制设备符合国标 + 仅限业余频段；3. 禁止用非业余频段设备|
|测试发射安全（0581/0580）|1. 选空闲 / 非热点频率；2. CW 方式按电键或 FM 不对话筒；3. 测试后恢复 SSB 方式|
|天线架设安全（0662）|1. 远离市电线路；2. 35kV 及以下电压安全距离≥1 米；3. 防止掉落接触供电装置|
|电源使用安全（0927/0973）|1. 电源内阻影响输出电压；2. 反接保护：串联熔断器 + 反向二极管；3. 过流 / 反接自动切断|
|继电器使用安全（1028/1029/1030）|1. 关断瞬间防反向电动势（并联电容 / 二极管）；2. 触点并联电容消火花干扰；3. 关注触点耐压 / 电流|

## 三、频率与干扰安全类

|题目核心|答案速记提纲|
|---|---|
|频率使用原则（0163/0174）|1. 核准频段内平等使用权；2. 次要业务不得干扰主要业务；3. 不擅自占用他人频率|
|干扰处理规范（0272）|1. 遇违法 / 不明干扰：不予理睬；2. 收集信息向无线电管理机构举报；3. 禁止压制干扰|
|禁止产生有害干扰（0133/0148）|1. 不得干扰合法业务；2. 干扰人身安全频率（船舶 / 航空器）：20-50 万罚款；3. 拒不改正：没收设备 + 吊销执照|

## 四、应急与特殊场景安全类

|题目核心|答案速记提纲|

|遇险信号处理（0269）|1. 立即确认险情 + 需求；2. 联系应急救援团队；3. 优先保障遇险通信|

## 六、电磁兼容（EMC）与干扰防护

1. **家庭环境防护**（新增 0676 题）
    
    - 短波电台与电视、无线路由器保持≥3 米距离，馈线远离市电线路
    - 天线架设需远离高压变压器（≥10 米），避免电磁耦合干扰
2. **特殊频段避让**（新增 0677 题）
    
    - 严禁在 118-137MHz（航空频段）、156-162MHz（航海频段）附近 50kHz 范围内发射，避免谐波落入
    - 发射前需用频谱仪检测 2 次谐波（如 28MHz 发射时检查 56MHz 附近），确保衰减≥40dB
